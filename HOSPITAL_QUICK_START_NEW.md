## üéØ Common Workflows

### Workflow 1: Emergency Situation
```
1. Patient arrives needing urgent blood
2. Go to Emergency page
3. Fill blood type and units
4. Send emergency alert
5. Wait for donor/blood bank responses
6. Check notifications for updates
```

### Workflow 2: Scheduled Surgery
```
1. Surgery scheduled, blood needed
2. Go to Patient Needed Blood
3. Fill patient details
4. Select best blood bank (check reputation)
5. Submit request
6. Blood bank receives notification
7. When blood arrives, mark as "Received"
8. Record automatically appears in Patient Records
```

### Workflow 3: Monthly Review
```
1. Go to Patient Records
2. Filter by date range
3. Review all transactions
4. Export report for management
5. Check Hospital Profile for statistics
```
