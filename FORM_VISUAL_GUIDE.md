## 🎯 Accessibility Features

### **Keyboard Navigation**
```
Tab Order:
1. Name field
2. Age field
3. Gender dropdown
4. Blood Group (disabled, skipped)
5. Occupation field
6. Phone field
7. Email field
... (continues through all fields)
N. Submit button

Enter Key: Submits form
Escape Key: Clears focus
Space Key: Toggles checkboxes
Arrow Keys: Navigate dropdown options
```

### **Screen Reader Support**
```
- All inputs have associated labels
- Required fields marked with * and aria-required="true"
- Checkboxes have descriptive labels
- Error messages announced
- Success/failure messages announced
- Section headers provide context
```

### **Focus Indicators**
```
Focused Input:
- Border: 2px solid red-600
- Ring: 2px red-600 with offset
- Outline: None (custom ring used)

Focused Checkbox:
- Ring: 2px red-600
- Outline: None

Focused Button:
- Ring: 2px red-600 with offset
- Outline: None
```
